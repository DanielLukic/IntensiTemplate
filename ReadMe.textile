Template for IntensiGame/IntensiBuild/RunME based games/applications.

h1. Dependencies

 * linux like environment

 * git 1.6 or later
 * ruby 1.8 or later
 * rake 0.8.3 or later
 * jruby 1.0 or later
 * java 1.5 or later
 * ant 1.5 or later
 * wtk 2.5 or later (for running the application in the emulator)

JRuby is used for the font sizer script. You may want to install rake and rspec within JRuby, too.
